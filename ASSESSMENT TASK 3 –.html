Source

Learn more about SQL Injection
Phishing

Source

Phishing is a fraudulent attempt to obtain sensitive information such as usernames, passwords, and credit card details by disguising oneself as a trustworthy entity in electronic communication.

Learn more about Phishing
Man-in-the-Middle (MitM) Attack

Source

A Man-in-the-Middle (MitM) attack occurs when an attacker secretly intercepts and possibly alters communications between two parties without their knowledge.

Learn more about MitM Attacks
Denial of Service (DoS) Attack

Source

A Denial of Service (DoS) attack aims to disrupt the normal functioning of a targeted server, service, or network by overwhelming it with a flood of internet traffic.

Learn more about DoS Attacks
Man-in-the-Browser (MitB) Attack

Source

A Man-in-the-Browser (MitB) attack is a type of cyber attack where the attacker installs malware on a victim's web browser to intercept and manipulate web transactions, potentially compromising sensitive information.

Learn more about MitB Attacks
Ransomware

Source

Ransomware is a type of malicious software that encrypts a victim's files and demands payment, usually in cryptocurrency, in exchange for decrypting them. It can cause significant data loss and financial damage.

Learn more about Ransomware
Cross-Site Request Forgery (CSRF)

Source

Cross-Site Request Forgery (CSRF) is a type of attack where a malicious website tricks a user's web browser into performing actions on another website where the user is authenticated, without the user's knowledge or consent.

Learn more about CSRF Attacks
DNS Spoofing (or DNS Cache Poisoning)

Source

This attack involves corrupting the DNS data stored on a DNS server's cache. Attackers manipulate this data to redirect users to malicious websites, often used in conjunction with phishing attacks.

Learn more about DNS Spoofing
BIOS/UEFI Firmware Attacks

Source

Attacking the system firmware (BIOS or UEFI) can provide attackers with persistent and low-level access to a system. Malicious firmware modifications can be used to evade detection, maintain persistence, or sabotage system functionality.

Learn more about BIOS/UEFI
